A. Quick Setup - top

There is no setup. Just open the SCPlaylistsPlayer.xcodeproj and you will be on your way. One prerequisite is that you have Xcode 6.2 installed on you mac.

B. Basic Configuration - top

The only thing you need to change the customize SoundcloudConfig.h file. (Code/SCPlaylistsPlayer/SCData/SoundcloudConfig.h)

C. Advanced Configuration - top

Adjustments
Add to your project
[1] Open your Xcode project - add (Edit/Add files "YourProject name") the SCData folder to your project. (Code/SCPlaylistsPlayer/SCData)
					
[2] Copy and paste to your project in Storyboard files (Playlists Controller - Playlist Controller).

[3] Connect the PlaylistsController to your storyboard.

[4] Customize SoundcloudConfig.h file.   

[5] App Soundcloud HTTP JSON API based. Portrait mode support, autolayout not support.
